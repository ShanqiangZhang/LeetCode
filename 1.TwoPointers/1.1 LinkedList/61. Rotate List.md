```java
题目: 
    rotate linkedlist by k
考点:
    1.Corner case:
        (1)不用rotate的情况 head为1 或null
        (2)list为null不能报空指针错
        (3)k > len 要取模处理
解法:
    1.快慢指针
    2.移动node要确保next被摁住

快满指针:
class Solution {
    public ListNode rotateRight(ListNode head, int k) {
        //1. get len 
        ListNode dummy = new ListNode(-1, head), p1 = head, tail = dummy;
        int len = 0;
        while (p1 != null) {
            len++;
            tail = p1;
            p1 = p1.next;
        }
        //2. corner case len <= 1 or k % len == 0
        if (len <= 1) return head;
        k %= len;
        if (k == 0) return head;
        
        //3. fast and slow pointer
        ListNode pre = dummy, slow = dummy, fast = dummy;
        // 3.1 move fast pointer k steps
        while (k-- > 0) {
            fast = fast.next;
        }
        // 3.2 get correct position
        while (fast != null) {
            pre = slow;
            slow = slow.next;
            fast = fast.next;
        }
        // 3.3 move last k list to front
        tail.next = dummy.next;
        dummy.next = slow;
        pre.next = null;
        
        return dummy.next;
    }
}

时空复杂度
Time: 获取长度n, 快慢指针n, 所以总共是O(n)
Space: 没有用额外空间 O(1)
```
